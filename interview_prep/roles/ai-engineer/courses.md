# AI Engineer: Free Courses

The best free courses for this role, grouped by topic. This repository's own courses come first because they are self-contained and built for practitioners; verified external courses follow. Every external link was checked and returned HTTP 200 at the time of writing. You do not need all of these: pick one per weak area and build alongside it.

## Agents and tool use

This repository's originals, ideal because agents are the highest-weighted 2025-2026 interview topic:

- [Agentic AI Crash Course](../../../free_courses/agentic_ai_crash_course/README.md): a 10-part crash course covering what agents are, the types of agentic systems, tools, RAG and agentic RAG, MCP, planning and reasoning models, memory, and multi-agent systems. This maps almost one to one onto the agent questions you will get.
- [OpenClaw Mastery for Everyone](../../../free_courses/openclaw_mastery_for_everyone/README.md): hands-on building with an agent harness.
- [Path: Agent Builder](../../../paths/agent-builder.md) and [Path: Harness Engineering](../../../paths/harness-engineering.md): sequenced learning paths through building agents.

Verified external:

- [Hugging Face Agents Course](https://huggingface.co/learn/agents-course) (2025): a free, current, hands-on course on building agents, tool use, and evaluation. Builds a working agent from fundamentals.
- [Anthropic Courses](https://github.com/anthropics/courses): free courses including tool use and building with an agent harness.

## Evaluation

Evaluation is repeatedly named the single biggest skill gap in AI Engineer candidates, so invest here.

- [AI Evals for Everyone](../../../free_courses/ai_evals_for_everyone/README.md): this repository's course on building evaluation for LLM systems, from defining success to LLM-as-judge to production monitoring. Builds the exact judgment interviewers probe.
- [Evaluation topic page](../../../topics/evaluation.md): the curated reading and video list to go with it.

## Prompting

- [Prompting topic page](../../../topics/prompting.md): the repo's curated prompting material, organized by level.
- [Prompt Engineering Guide](https://www.promptingguide.ai/) (dair-ai, kept current): a free, thorough reference course on prompting techniques. Builds a working vocabulary of patterns and when to use each.

## Retrieval and RAG

RAG is the most common take-home and a core system-design topic, so build at least one real RAG app.

- [RAG topic page](../../../topics/rag.md): the repo's full curated RAG course list plus the [RAG Roadmap](../../../resources/RAG_roadmap.md) and [Agentic RAG 101](../../../resources/agentic_rag_101.md).
- [Retrieval Augmented Generation (DeepLearning.AI)](https://www.deeplearning.ai/courses/retrieval-augmented-generation) (2025): a free course on building and evaluating RAG systems end to end. Builds chunking, retrieval, and evaluation intuition.
- [RAG Techniques](https://github.com/NirDiamant/RAG_Techniques): a course-like collection of runnable RAG notebooks from basic to advanced. Builds hands-on pattern coverage.

## End-to-end AI engineering

Broad courses that cover the whole role, good for filling gaps fast:

- [Generative AI Genius](../../../free_courses/generative_ai_genius/README.md): this repository's broad generative AI course.
- [Applied LLMs Mastery 2024](../../../free_courses/Applied_LLMs_Mastery_2024/README.md): a repo course spanning prompting, RAG, fine-tuning, evaluation, and deployment (2024, aging but still a solid map).
- [Generative AI for Beginners](https://github.com/microsoft/generative-ai-for-beginners) (Microsoft, kept current): a free, structured, hands-on course across the whole GenAI stack. Builds breadth with runnable lessons.
- [Hugging Face LLM Course](https://huggingface.co/learn/llm-course) (2025): transformers, tokenization, using and adapting models. Builds fundamentals with code.

## Fine-tuning (adjacent depth)

You mostly consume models, but expect a few fine-tuning questions and know when it is the right tool.

- [Fine-tuning topic page](../../../topics/fine-tuning.md) and [Fine-tuning 101](../../../resources/fine_tuning_101.md): the repo's curated material.
- [smol-course](https://github.com/huggingface/smol-course) (Hugging Face, current): a free, practical course on aligning and fine-tuning small models. Builds the SFT, DPO, and evaluation loop hands-on.
- [LLM Course](https://github.com/mlabonne/llm-course) (mlabonne, kept current): a widely used roadmap plus notebooks covering fundamentals, fine-tuning, and deployment.

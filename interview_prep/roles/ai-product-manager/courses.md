# AI Product Manager: Free Courses

The best free courses for AI PM prep, grouped by topic. Start with this repository's own courses, which are written for practitioners and PMs, then use the external ones to fill gaps. Every external link was checked and returned HTTP 200. Free means free to access or free to audit.

For the full free-course catalog by topic, see the repo's [courses.md](../../../courses.md).

---

## Start here: this repository's own courses

- **[AI Evals for Everyone](../../../free_courses/ai_evals_for_everyone/README.md)**: builds the exact skill the metrics round tests. Offline eval sets, LLM-as-judge, faithfulness, online monitoring, and how to define a quality bar. Do this one first; the evaluation round is the round that most predicts the hire.
- **[Agentic AI Crash Course](../../../free_courses/agentic_ai_crash_course/README.md)**: builds an intuition for what agents are, when to use one, and how they fail. Directly feeds the technical-literacy and agent-design questions.
- **[Applied GenAI and other courses (repo catalog)](../../../courses.md)**: the full grouped list of free courses maintained in this repo.

## AI foundations for PMs (what models can and cannot do)

- **[Generative AI for Beginners (Microsoft)](https://github.com/microsoft/generative-ai-for-beginners)**: 21 lessons covering LLMs, prompting, RAG, embeddings, and building apps. The best broad, free, hands-on grounding for a non-engineer PM. Builds end-to-end vocabulary.
- **[AI for Beginners (Microsoft)](https://github.com/microsoft/AI-For-Beginners)**: a 12-week curriculum on AI broadly (classic ML through neural nets). Use it to shore up the classic-ML-versus-LLM judgment.
- **[Machine Learning Crash Course (Google)](https://developers.google.com/machine-learning/crash-course)**: fast conceptual grounding in ML fundamentals, useful for the model-selection and metrics questions. Foundational anchor.
- **[DeepLearning.AI short courses](https://www.deeplearning.ai/short-courses/)**: 1-to-2 hour free courses on RAG, agents, evaluation, prompting, and function calling. Pick the ones matching your weak areas from [questions.md](questions.md). Builds targeted depth fast.

## Agents (the fastest-growing area interviewers probe)

- **[AI Agents for Beginners (Microsoft)](https://github.com/microsoft/ai-agents-for-beginners)**: 11 lessons on agent design patterns, tool use, planning, and multi-agent systems. Builds the agent-product judgment the Sierra and OpenAI style loops test.
- **[Hugging Face Agents Course](https://huggingface.co/learn/agents-course)**: a hands-on course on building and evaluating agents, including tools, memory, and agent evaluation. Builds practical agent literacy.
- **[Anthropic Courses](https://github.com/anthropics/courses)**: free courses on prompt engineering, tool use, and building with models, straight from a frontier lab. Builds prompting and tool-use fluency.
- Pair with the repo's [Agent Builder path](../../../paths/agent-builder.md) and [Harness Engineering path](../../../paths/harness-engineering.md).

## Evaluation and production

- **[AI Evals for Everyone (this repo)](../../../free_courses/ai_evals_for_everyone/README.md)**: as above, the anchor course for the metrics round.
- **[DeepLearning.AI catalog](https://www.deeplearning.ai/courses/)**: browse for the current free short courses on evaluating and monitoring LLM apps and building RAG. Builds offline-and-online eval skills.
- Pair with the repo's [Production topic](../../../topics/production.md) and [Evaluation topic](../../../topics/evaluation.md).

## The PM craft (product sense, metrics, execution)

- **[Managing Machine Learning Projects (Google)](https://developers.google.com/machine-learning/managing-ml-projects)**: Google's free guidance on scoping and running ML projects as a PM. Builds the ML-project-management and expectation-setting muscle.
- **[Product Sense interview prep (Exponent)](https://www.tryexponent.com/blog/product-sense-interview)**: the classic product-sense frame the AI product-sense round is built on top of. Free blog guide.
- **[People + AI Guidebook (Google PAIR)](https://pair.withgoogle.com/guidebook/)**: not a course but a structured, chapter-by-chapter guide to designing AI UX. Work through it like one; it directly powers the product-sense round.

## How to sequence the courses

1. If short on time, do only **AI Evals for Everyone** and **Generative AI for Beginners**, then read the [Anthropic agent guides in resources.md](resources.md).
2. With more time, add **AI Agents for Beginners** or the **Hugging Face Agents Course**, plus 2 or 3 targeted **DeepLearning.AI short courses** on your weakest themes.
3. Throughout, keep the [People + AI Guidebook](https://pair.withgoogle.com/guidebook/) open for the design-UX questions.

See the day-by-day sequencing in [prep-plan.md](prep-plan.md).

# The real-model path is provider-agnostic (LangChain init_chat_model). To use a real model,
# install the integration for your provider, then set VOICE_AGENT_MODEL and the matching key:
#   OpenAI:    pip install langchain-openai        VOICE_AGENT_MODEL=gpt-realtime       OPENAI_API_KEY=...
#   Anthropic: pip install langchain-anthropic     VOICE_AGENT_MODEL=claude-sonnet-5    ANTHROPIC_API_KEY=...
#   Gemini:    pip install langchain-google-genai  VOICE_AGENT_MODEL=gemini-2.0-flash   GOOGLE_API_KEY=...
# With no key set, the agent runs on the deterministic offline policy (no provider needed).

# A real voice system also needs streaming ASR and TTS (or a single speech-to-speech model)
# around this control-flow graph. Those are out of scope for this offline demo; see the case study.

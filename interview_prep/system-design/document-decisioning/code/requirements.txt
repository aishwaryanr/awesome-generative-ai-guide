# The real-model path is provider-agnostic (LangChain init_chat_model). To route the
# borderline judgment through a real model, install the integration for your provider,
# then set UW_AGENT_MODEL and the matching key:
#   OpenAI:    pip install langchain-openai        UW_AGENT_MODEL=gpt-4o-mini      OPENAI_API_KEY=...
#   Anthropic: pip install langchain-anthropic     UW_AGENT_MODEL=claude-sonnet-5  ANTHROPIC_API_KEY=...
#   Gemini:    pip install langchain-google-genai  UW_AGENT_MODEL=gemini-2.0-flash GOOGLE_API_KEY=...
# With no key set, the agent runs on the deterministic offline policy (no provider needed).

# field extraction, install the integration for your provider, then set FINOPS_MODEL and the key:
#   OpenAI:    pip install langchain-openai        FINOPS_MODEL=gpt-4o-mini      OPENAI_API_KEY=...
#   Anthropic: pip install langchain-anthropic     FINOPS_MODEL=claude-sonnet-5  ANTHROPIC_API_KEY=...
#   Gemini:    pip install langchain-google-genai  FINOPS_MODEL=gemini-2.0-flash GOOGLE_API_KEY=...
# With no key set, the agent runs on the deterministic offline parser (no provider needed).

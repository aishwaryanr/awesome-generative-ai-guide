#   OpenAI:    pip install langchain-openai        COMPLIANCE_AGENT_MODEL=gpt-4o-mini      OPENAI_API_KEY=...
#   Anthropic: pip install langchain-anthropic     COMPLIANCE_AGENT_MODEL=claude-sonnet-5  ANTHROPIC_API_KEY=...
#   Gemini:    pip install langchain-google-genai  COMPLIANCE_AGENT_MODEL=gemini-2.0-flash GOOGLE_API_KEY=...
# With no key set, the agent runs on the deterministic offline policy (no provider needed).

# Path: Agent Builder

A curated sequence through the Build journey toward one outcome: shipping a real agentic system. Every step below is existing repo content, ordered so each builds on the last. This is a named path inside [Build AI](../journeys/build.md), not a separate course.

**Tags:** `Level 🟢/🟡/🔴 · Format 📖 Read / 🎥 Video / 🛠️ Notebook · Source ⭐ LevelUp Labs original · year`.

---

## The sequence

1. **[Agentic AI Crash Course](../free_courses/agentic_ai_crash_course/README.md)** ⭐ 📖 (2025) 🟢 Build 101: understand agents, tools, RAG, MCP, planning, memory, and multi-agent systems. Start here for the vocabulary and the mental model.
2. **[Agentic RAG 101](../resources/agentic_rag_101.md)** ⭐ 📖 (2024) 🟡 Build 201: give the agent grounded retrieval by combining RAG with agentic control.
3. **[Agentic AI Notebooks (LinkedIn Learning)](../resources/agentic_ai_course_lil/README.md)** ⭐ 🛠️ (2026) 🟡 Build 201: get hands-on with action autonomy and planning autonomy in runnable notebooks.
4. **[AI Evals for Everyone](../free_courses/ai_evals_for_everyone/README.md)** ⭐ 📖 (2026) 🟡 Build 201: measure whether the agent actually works, with evaluation and monitoring.
5. **[Securing Agentic AI Systems](../resources/securing_agentic_ai_systems.md)** ⭐ 📖 (2026) 🔴 Build 301: harden the system against agentic attack vectors before it ships.

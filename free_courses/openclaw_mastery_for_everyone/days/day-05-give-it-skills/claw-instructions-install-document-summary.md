# Day 5: Install `document-summary`

Install `document-summary` into this workspace.

If the user has not already approved the install, ask for confirmation first.
If it is already installed and ready, report that instead of duplicating it.

After install, tell the user:

- where the skill lives
- that they should type `/new` in OpenClaw before trying to use the new skill
- the exact message to test it

Use a clear test pattern such as:

- `Summarize this text: <content>`
- `Use document-summary on this link: <url>`

Stop when the install report is complete.
